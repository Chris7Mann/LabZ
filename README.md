# LabZ // Infrastructure & Security Portfolio

Questo repository ospita il codice sorgente del mio portfolio personale, progettato come un'estensione del mio laboratorio IT. Il sito documenta il mio percorso di apprendimento, i progetti di rete, la gestione di sistemi e l'implementazione di soluzioni di sicurezza.

## // TECH_STACK
Il progetto è costruito utilizzando le tecnologie web più moderne per garantire performance e una user experience pulita:

* **Framework**: [Next.js 15](https://nextjs.org/) (App Router)
* **Linguaggio**: TypeScript
* **Styling**: Tailwind CSS
* **Deployment**: [Vercel / Cloudflare]
* **Gestione Privacy**: CookieScript (Compliance GDPR)
